# Negotiation Skill — 협상 설계 엔진

person-profiler가 "상대를 아는 것", biz-skill이 "전략을 짜는 것"이라면, 이 스킬은 **"테이블 위에서 어떻게 주고받을 것인가"를 5-Layer 깊이로 구조화하는 것**.

협상은 준비의 질이 결과의 80%를 결정한다(HBS PON). 이 스킬은 그 준비를 프로토콜화하되, 심리학·게임이론·문화차이·프로세스 설계를 체계적으로 통합한다.

---

## ⛔ 절대 규칙

| # | 규칙 | 이유 |
|---|------|------|
| 1 | **양측 BATNA 미확정 상태로 전략 도출 금지** | BATNA가 협상력의 기반. 미확정=기반 없는 전략 |
| 2 | **단일 시나리오 금지** — 최소 3시나리오(최선·기본·최악) 필수 | 단일 시나리오=준비 안 한 것 |
| 3 | **상대방 관점 생략 금지** — 모든 분석에 상대방 이해관계·제약·BATNA 병치 | 일방적 분석=실전에서 무력화 |
| 4 | **심리 메커니즘 스크리닝 필수** — 상대방이 어떤 심리 트리거(앵커링/반응 평가절하/손실회피)에 약한지 판정 | 일반적 전술 ≠ 개인화된 전술 |
| 5 | **게임이론 체크** — 반복게임인가? 정보비대칭인가? 약속 가능한가? | 전술을 게임 구조에 정렬해야 지속 가능 |
| 6 | **윤리 경계 명시 필수** — 기만·조작은 리스크 경고와 함께만 제시 | 장기 관계 파괴 리스크. 평판 보호 |

## §1. 모드 자동 판별

| 입력 유형 | 모드 | 예시 |
|---|---|---|
| 앞으로의 협상 상황 | **설계** | "이번 텀시트 협상 어떻게 준비하지" |
| 진행 중 교착 | **돌파** | "협상이 막혔어", "상대가 안 움직여", "교착 상황" |
| 완료된 협상 | **복기** | "이번 딜 결과 분석해줘" |

복합 시 설계→돌파 파이프라인. 애매하면 명확화 질문.

## §2. 5-Layer Analysis Architecture

협상의 성패는 5개 층이 동시에 작동할 때 결정된다. 각 층은 독립적이지만 상호 제약한다.

| Layer | 핵심 질문 | 출처 | 스크리닝 |
|-------|----------|------|---------|
| **L1: Interest** | 양측의 진짜 이해관계·동기·우선순위는? Position vs Interest 분리 가능한가? | Fisher & Ury, Bazerman 7원리 | ● 필수 / ○ 관련 / - 무관 |
| **L2: Structure** | BATNA·ZOPA·제안 공간을 정의하는 구조는? 3-D 협상(Setup·Design·Tactics) 프레임에서 어느 차원이 가장 영향력 있는가? | Lax & Sebenius, Raiffa | ● 필수 / ○ 관련 / - 무관 |
| **L3: Psychology** | 상대방을 지배하는 심리 메커니즘은 무엇인가? (앵커링·반응 평가절하·손실회피·공정성 규범 중) | Tversky, Kahneman, Cialdini | ● 필수 / ○ 관련 / - 무관 |
| **L4: Relationship** | 문화적 맥락·권력 역학·신뢰 수준이 어떤 제약을 만드는가? | Hofstede, Meyer, Shadow Negotiation | ● 필수 / ○ 관련 / - 무관 |
| **L5: Execution** | 합의 후 이행 구조·조건부 계약·프로세스 설계는? | Raiffa, Game Theory Folk Theorem | ● 필수 / ○ 관련 / - 무관 |

**Layer 상호 작용 규칙**:
- L1 우선: Interest를 구명하지 않고 L2 구조 설계 금지
- L3는 L2 전술 선택을 통제: 앵커링 약한 상대에게는 앵커링 전술 無효
- L4는 L3 심리 기법을 제약: 문화적 비위험 회피 문화에서 강한 앵커링은 관계 파괴
- L5는 모든 층을 구현: 아무리 좋은 전략도 이행 구조가 없으면 무의미

## §3. 심리 프로필 스크리닝

상대방이 어떤 심리 메커니즘에 가장 강하게 반응하는가를 판정 → 전술 개인화.

| 심리 메커니즘 | 신호 | 전술 응답 |
|---|---|---|
| **앵커링 취약** | 첫 제안에 "그게 우리 가격이 맞아?"라고 선뜻 따라감 | 공격적 앵커링 활용 + 근거 제시 (약함) |
| **반응 평가절하** | "상대가 제안했다고 좋은 건 아니야"라는 고정관념 반복 | 제3자 재제시 또는 근거-우선 프레이밍 |
| **손실회피 강함** | "현재 상태 유지"를 강하게 선호, 변화 제안 거부 | 손실 프레이밍 ("지금 안 하면 기회상실") |
| **공정성 규범 민감** | "같이 50:50 나눠야지"라는 표현 빈번 | 공정성 기준을 먼저 합의하고 적용 |
| **지위·위신** | 자신의 위치·권력 표시 빈번 | 상대 체면 보장 장치 (숨겨진 양보 전술) |

→ 분석 깊이: references/negotiation-psychology.md

## §4. 게임이론 의무 체크

모든 협상 전략을 게임이론 렌즈로 재검토. 3가지 핵심 질문:

| 질문 | 찾는 것 | 전술 조정 |
|------|--------|---------|
| **반복게임인가?** | 일회성 vs 지속 관계 | 반복 → 협력 인센티브 강화 / 일회성 → 배팅 전술 OK |
| **정보 비대칭?** | 누가 더 많이 알고 있는가 | 정보 공개 전술, 조건부 계약 활용 |
| **약속 가능한가?** | 상대방 약속 신뢰도, 제3자 집행 메커니즘 | 약속 불가 → 구속력 있는 조건부 계약 필수 |

→ 분석 깊이: references/game-theory.md

## §5. 실행 파이프라인

### 설계 모드

```
①상황 구조화 → ②5-Layer 스크리닝 → ③심리 프로필 분석 → ④게임이론 검증 
→ ⑤Deep Layer 분석(활성화된 층만) → ⑥전략 설계 → ⑦3-시나리오 시뮬레이션 → ⑧액션 플랜
```

**①상황 구조화**: 협상 대상·상대방·쟁점·배경·제약·시간압박·관계 맥락 추출.

**②5-Layer 스크리닝** (각 층을 ● 핵심 / ○ 관련 / - 무관으로 1줄 판정):

| Layer | 판정 | 핵심 질문 |
|-------|------|---------|
| L1 Interest | ● / ○ / - | |
| L2 Structure | ● / ○ / - | |
| L3 Psychology | ● / ○ / - | |
| L4 Relationship | ● / ○ / - | |
| L5 Execution | ● / ○ / - | |

**③심리 프로필 분석** (●로 판정된 층에서):
- 상대방의 주요 심리 트리거 (앵커링/반응평가절하/손실회피/공정성/지위)
- 각 트리거의 강도 (강·중·약)
- 이에 따른 전술 선택

**④게임이론 검증**:
- 반복게임 여부 + 할인인수 추정 (얼마나 미래를 중시하는가)
- 정보 비대칭 구조 (누가 뭘 숨기고 있는가)
- 약속 실행 메커니즘 (자발적 vs 구속력 있는 계약)

**⑤Deep Layer 분석** (활성화 층만):
- L1 ● → 이해관계 매핑 (→ references/negotiation-theories.md §Fisher & Ury)
- L2 ● → BATNA·ZOPA·3-D 협상 (→ references/advanced-frameworks.md)
- L3 ● → 심리 전술 설계 (→ references/negotiation-psychology.md)
- L4 ● → 문화·권력 분석 (→ references/cultural-negotiation.md)
- L5 ● → 프로세스 설계 (→ references/process-design.md)

**⑥전략 설계**:

| 전략 요소 | 설계 내용 |
|-----------|----------|
| BATNA·ZOPA 매핑 | 양측 대안·유보가격·목표점·앵커 |
| 앵커링 전술 | 첫 제안 금액/조건 + 근거 프레이밍 (상대 심리 프로필 기반) |
| 양보 시퀀스 | 양보 항목·교환 비율·양보 곡선 (체감형) |
| 패키지 딜 | 로그롤링 (다중 쟁점 번들) |
| 감정·관계 관리 | 체면 보장 장치·문화적 민감성 |

**⑦3-시나리오 시뮬레이션**:

| 시나리오 | 상대 반응 | 우리 대응 | 예상 결과 | L3 심리 기전 |
|----------|---------|----------|-----------|-------------|
| 최선 | | | | |
| 기본 | | | | |
| 최악 | | | | |

각 시나리오에 If-Then 매핑. L3 심리 메커니즘이 반응을 어떻게 예측 가능하게 하는지 명시.

**⑧액션 플랜**: 협상 전·중·후 체크리스트 + 응급 대응.

### 돌파 모드 — Deadlock Diagnosis Matrix

교착이 발생했을 때, 5-Layer × 5-Pattern 기반 진단.

| 교착 패턴 | L1 Interest | L2 Structure | L3 Psychology | L4 Relationship | L5 Execution |
|----------|-----------|-----------|-------------|------------|----------|
| **Position 고착** | Interest 미탐색 | ZOPA 부재 | 고정된 앵커 | 신뢰 부족 | - |
| **정보 비대칭** | 숨겨진 우선순위 | 불투명 대안 | 의도 의심 | 신뢰 결핍 | - |
| **감정 에스컬레이션** | - | - | 손실 프레이밍 고착 | 관계 악화 | - |
| **체면·권위 문제** | - | - | 지위 경쟁 | 문화적 권력 역학 | - |
| **약속 신뢰도** | - | - | - | - | 실행 메커니즘 부재 |

각 셀에서 진단 신호 → 권장 기법 → 예상 결과.

### 복기 모드

①결과 구조화(합의 vs 목표) → ②5-Layer별 평가 → ③각 Layer에서 뭘 배웠는가 → ④다음 협상 개선점.

## §6. 협상 유형별 어댑터

| 파트너십/제휴 | 장기관계·상호의존 | L1(관심), L4(관계) | biz-skill |

## Gotchas

| 함정 | 대응 |
|------|------|
| BATNA 없이 전략 짜기 | §5 ①에서 양측 BATNA 미확정 → STOP + 명확화 |
| 5-Layer 중 1개만 고집 | 5개 층을 모두 스크리닝. 활성화된 층만 깊이 분석 |
| 상대방 심리 프로필 건너뛰기 | L3 ●으로 판정되면 §3 스크리닝 필수 |
| "게임이론은 복잡해"라며 체크 스킵 | §4의 3가지 질문만 답하면 됨. 수학 불필요 |
| 전술만 짜고 프로세스 설계 무시 | L5 무시 = 합의 후 이행 파괴. §5⑧에서 최소 액션 플랜 필수 |
| 단일 시나리오 | 최선·기본·최악 3가지. 각 시나리오에 L3 심리 기전 연결 |
| 문화적 맥락 무시 | L4 관계 분석에서 상대의 문화 프로필 필수 (→ references/cultural-negotiation.md) |
| 상대를 "적"으로만 프레이밍 | L1 Interest에서 공동 이익 영역 반드시 탐색 |
| "이번엔 강하게 나가자" 감정 결정 | 모든 전술을 L3·L4를 통해 재검증. 감정은 정보 아님 |
