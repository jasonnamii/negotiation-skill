# Cultural Negotiation — 문화적 협상 차이

## Hofstede 6 Cultural Dimensions 협상 적용

### 6가지 차원별 분석

#### 1. Power Distance (권력거리)

| 차원 | 정의 | 높음(High PD) | 낮음(Low PD) |
|------|------|---------|---------|
| **정의** | 권력의 불평등을 얼마나 수용하는가? | 계층 구조, 상하관계 명확 | 평등, 위계 최소화 |
| **국가 사례** | | 한국, 일본, 인도, 말레이시아 | 노르웨이, 덴마크, 오스트리아, 미국 |
| **협상 영향** | | CEO만 결정권. 담당자는 "상급자 결재 필요"라고 주장 | 담당자도 결정 가능. 의견 평등 존중 |
| **협상 전략** | | 권력자(CEO)의 신뢰 먼저 확보. 하급자 무시 금지 (평판 손상) | 담당자와 직접 협상 가능. 계층 표시 금지 |
| **주의** | | 상급자 없이 결정 강요 금지 → 협상 파기 | 과도한 형식 → 비효율성 느낌 |

#### 2. Individualism vs Collectivism (개인주의 vs 집단주의)

| 차원 | 높음(Individual) | 낮음(Collectivism) |
|------|---------|---------|
| **국가** | 미국, 호주, 영국, 네덜란드 | 한국, 중국, 일본, 멕시코 |
| **협상 목표** | 개인/회사 이익 극대화. "우리가 얼마나 이득인가" | 집단 이익, 장기 관계, 상호 번영. "함께 성공하는가" |
| **의사결정** | 빠름 (개인 판단) | 느림 (합의 도출 필요) |
| **신뢰 구축** | 논리·근거 제시. 합리적 이점 강조 | 관계 첫 구축. 밥 함께 먹기, 개인적 공감 필요 |
| **양보 태도** | "이 거래에서만 최대한 이득" | "향후 거래도 고려해서 상호 이득" |
| **협상 전술** | 강한 입장. 하지만 이성적 설득 | 부드러운 입장. 하지만 관계 무시 금지 |

#### 3. Masculinity (경쟁성/성취)

| 차원 | 높음(Masculine) | 낮음(Feminine) |
|------|---------|---------|
| **국가** | 일본, 독일, 오스트리아, 홍콩 | 스웨덴, 노르웨이, 네덜란드, 덴마크 |
| **협상 목표** | "우리가 이기는 거" (분배 극대화) | "함께 풀기" (통합적 솔루션) |
| **경쟁 태도** | 경쟁을 선호. "이 거래를 놓치면 경쟁사가 가져간다" | 경쟁보다는 공생. 모두 윈-윈 추구 |
| **감정 표현** | 절제. 감정은 약함의 신호 | 공감 표현. 상대의 어려움 인정 |
| **결과 기준** | 수치 중심 (매출, 수익률) | 만족도 중심 (관계 품질, 장기성) |
| **협상 전술** | 강한 압박, 데드라인 활용 | 포용적 접근, 상대의 우려 수렴 |

#### 4. Uncertainty Avoidance (불확실성 회피)

| 차원 | 높음(High UA) | 낮음(Low UA) |
|------|---------|---------|
| **국가** | 그리스, 벨기에, 일본, 프랑스 | 싱가포르, 덴마크, 홍콩, 미국 |
| **협상 목표** | 불확실성 최소화. 세밀한 계약 필수 | 빠른 합의. 세부사항은 나중 조정 |
| **계약서** | 두꺼운 계약 (모든 경우의 수 포함). 법률가 동원 | 얇은 계약 (핵심만). 신뢰 기반 |
| **변수 처리** | "만약 X가 발생하면?" 사전 정의 필수 | "그때 가서 의논하자" 가능 |
| **실행 감시** | 엄격한 모니터링, 분기별 리뷰 | 자율성 존중 |
| **협상 전술** | 구체적 기준·절차 미리 합의. "만약" 시나리오 연습 | 빠른 "In Principle" 합의 후 디테일 조정 |

#### 5. Long-term Orientation (장기 지향성)

| 차원 | 높음(Long-term) | 낮음(Short-term) |
|------|---------|---------|
| **국가** | 한국, 중국, 일본, 대만 | 미국, 영국, 호주, 파키스탄 |
| **협상 목표** | 관계 지속. "우리가 언제까지 함께하는가" | 거래 성사. "이번 거래가 얼마인가" |
| **시간 관점** | "5년, 10년 후 우리가 어떻게 될 것인가" | "이번 분기, 내년 성과가 뭔가" |
| **신뢰** | 신뢰 구축에 시간 필요. 한 번 깨지면 회복 어려움 | 거래 기반 신뢰. 계약으로 보호 |
| **가격 협상** | 초기 약간의 손실 감수 → 장기 이윤. "고객 평생가치" | 초기 수익 극대화 |
| **협상 전술** | 관계 투자 언급. "함께 성장하는 파트너" | 단기 인센티브. "이번 거래의 immediate benefit" |

#### 6. Indulgence (관용성/쾌락성)

| 차원 | 높음(Indulgent) | 낮음(Restrained) |
|------|---------|---------|
| **국가** | 멕시코, 브라질, 호주 | 홍콩, 러시아, 일본, 한국 |
| **협상 톤** | 친근한 톤, 유머, 관계 강조 | 형식적 톤, 절제, 업무 중심 |
| **휴식·여유** | 협상 중간 휴식, 밥 시간 중요 | 효율성 중심. 불필요한 시간낭비 금지 |
| **개인적 대화** | 가족, 취미 얘기 OK. 신뢰 구축 | 개인사 언급 금지. 전문성만 |
| **협상 진행** | 느슨한 일정. "되는 대로" | 타이트한 일정. 데드라인 준수 |

## Erin Meyer Cultural Map — 8가지 스케일

### 8가지 협상 영향 스케일

| # | 스케일 | 직선형 | 곡선형 | 협상 영향 |
|---|--------|--------|--------|---------|
| **1** | Communicating | 직접 (Tell) | 간접 (Hint) | 직접 스타일: 직설적 거부 OK. 간접 스타일: 거부를 완곡하게 표현. 해석 필요 |
| **2** | Evaluating | 명확한 피드백 (Good/Bad) | 암묵적 피드백 | 명확: "이 제안은 부족하다"고 직설. 암묵: "흥미롭네요"만 해도 거부의 신호 |
| **3** | Persuading | 원칙-우선 (Why) | 실제사례-우선 (How) | 원칙: 이론·근거부터. 실제사례: 사례·성공 스토리부터 |
| **4** | Leading | 평등적 (모두 의견 존중) | 계층적 (리더 주도) | 평등: 담당자 의견도 중요. 계층: 리더만 결정 |
| **5** | Deciding | 합의 기반 (모두 동의) | 리더 중심 (CEO 결정) | 합의: 느리지만 모두 수용. 리더: 빠르지만 하급자 불만 가능 |
| **6** | Trusting | 업무 기반 (Task trust) | 관계 기반 (Relationship trust) | 업무: 결과·능력으로 신뢰. 관계: 개인적 친분으로 신뢰 |
| **7** | Disagreeing | 대면적 (직접 의견 충돌) | 회피적 (조화 유지) | 대면: 직접 충돌해야 해결. 회피: 간접 방법 필요 |
| **8** | Scheduling | 선형 (순차적, 계획 준수) | 유연한 (동시다중, 유동적) | 선형: 일정 변경 금지. 유연: 상황에 따라 즉흥적 조정 |

### Meyer의 "No" 예제 (문화별 거부 신호)

**미국 (Direct)**: "No, that doesn't work for us." (명확한 거부)

**한국 (Indirect)**: "흠... 좋은 의견이네요. 한 번 검토해보겠습니다." (실제는 거부이지만 완곡)

**일본 (Very Indirect)**: "그 부분은 좀 어렵지 않을까 싶네요." (거부를 암시적으로)

→ **협상에서 상대의 "No"를 인식하지 못할 수 있다.** 명시적 확인 필수.

## Power Asymmetry 관리

### 권력의 출처 5가지

| 출처 | 정의 | 협상 영향 | 대응 |
|------|------|---------|------|
| **Punishment** (처벌 권력) | 거부 시 처벌 가능 (거래 중단, 법적 소송) | 강한 압박력. 상대가 두려움 | "처벌보다는 상호 이익" 강조. 신뢰 구축 |
| **Reward** (보상 권력) | 수용 시 보상 (더 높은 가격 지불, 우선 거래) | 긍정적 동기 유발 | 명확한 보상 명시. 실현 가능성 보증 |
| **Position** (지위 권력) | CEO, 선임자의 위치 | "나는 결정권이 있다"는 신호 | 상대의 지위도 존중. 계층 간 권력 균형 |
| **Knowledge** (정보 권력) | 시장 정보, 기술 정보를 독점 | "우리만 알고 있다"는 우월감 | 정보 투명성. 상대도 확인 가능하게 |
| **Admiration** (칭찬 권력) | 대상의 능력·성과를 존경 | 감정적 설득. 강함 | 상호 존경. "당신의 능력을 인정합니다" |

### 저권력자의 협상 전략

| 상황 | 전술 | 예 |
|------|------|------|
| **정보 권력 약함** | 제3자 검증 활용. 객관 기준 도입 | "이 가격이 공정한지 industry benchmark로 확인해주세요" |
| **지위 권력 약함** | 자신의 전문성·성과로 보상. 부하직원도 신뢰 구축 | "이 프로젝트는 제가 리드한 3개 성공 사례가 있습니다" |
| **대안 부족** | BATNA 강화. 새로운 대안 개발 | "다른 공급업체도 검토 중입니다" (거짓 아님) |
| **관계 약함** | 상대의 내부 champion 찾기. 공통 이해관계자 활용 | "CFO분은 우리의 비용 절감 효과에 관심 있지 않을까요?" |

### 권력 비대칭 상황의 협상 프로토콜

```
Step 1: 권력 원천 진단 (5가지 중 어디?)
Step 2: 약한 영역 보강 (정보, 지위, 대안, 관계 중 어디?)
Step 3: 제3자 활용으로 중립성 확보
Step 4: 상대의 내부 제약 파악 → 상대도 "이 협상 성공이 필요"하게 만들기
Step 5: 상호 이익 강조 (권력 차이 인정하되, 함께 성공하는 프레임)
```

## Principal-Agent Problem — 주인-대리인 문제

협상에서 상대방이 "담당자" = 주인의 이익과 다를 수 있다.

| 상황 | 문제 | 신호 | 대응 |
|------|------|------|------|
| **담당자가 보수적** | "상급자 승인 필요"라며 거부 반복 | 결재 권한 명확하지 않음. 회의만 반복 | 상급자와 직접 협상 요청. 프로세스 명확히 |
| **담당자가 과대 정책** | "이 조건으로 가능합니다"했는데, 나중에 상급자가 "안 된다" | 담당자의 권한 범위 불명확 | 사전에 "누가 최종 결정자인가" 확인. 위임 권한 서명받기 |
| **담당자 이기주의** | 담당자 개인의 보너스·평가만 중시. 회사 이익 무시 | "이 조건이면 제 보너스는?" | 회사 이익과 개인 이익을 동일시할 수 있는 조건 설계 |

### 대리인 문제 해결

1. **권한 명확화**: "당신이 최종 결정할 수 있는 범위가 정확히 뭔가요?"
2. **인센티브 정렬**: 개인 이익과 회사 이익을 함께 고려하는 구조
3. **상급자 조기 개입**: "상급자도 이 방향에 동의하시나요?"

## Cultural Diagnosis Quick Protocol

협상 상대의 문화 프로필을 5가지 질문으로 파악.

| # | 질문 | 응답 분석 |
|---|------|---------|
| **1** | "당신 회사에서는 중요한 결정을 누가 내리나요?" | 답: CEO → High Power Distance. 답: 팀 합의 → Low PD |
| **2** | "실패나 부정적 피드백을 어떻게 표현하나요?" | 직설적 → Direct. 완곡 → Indirect |
| **3** | "이 거래의 성공을 어떻게 정의하나요?" | "우리 수익" → Individualistic. "함께 성장" → Collectivistic |
| **4** | "장기적으로 우리 관계를 어떻게 보나요?" | "계약 기간만" → Short-term. "10년 파트너" → Long-term |
| **5** | "우리가 준비해야 할 계약 수준은 어느 정도인가요?" | "간단한 MOU" → Low UA. "상세한 계약서" → High UA |

→ 5개 답으로 상대의 Hofstede 6차원 + Meyer 8스케일 프로필 추론
