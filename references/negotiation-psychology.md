# Negotiation Psychology — 협상 심리 메커니즘

## Anchoring & Adjustment — 앵커링과 조정

### 메커니즘 (Tversky & Kahneman 1974)

**실험**: 
- A조: "휠 돌려서 나온 숫자(65)와 비교해서, 아프리카 UN 가입국 비율이 몇 %인가?" → 평균 45%
- B조: "휠 돌려서 나온 숫자(10)와 비교해서, 아프리카 UN 가입국 비율이 몇 %인가?" → 평균 25%

**실제 답**: 약 35%

→ **임의의 숫자도 판단을 왜곡한다.** 심지어 "이건 랜덤입니다"라고 설명해도 작동.

### 협상에서의 앵커링

| 상황 | 앵커 제시 | 효과 | 조정 과정 |
|------|----------|------|---------|
| **판매자 입장** | "이 제품은 $100만이 시장가입니다" (실제: $80만 가능) | 상대가 $80만을 "저가"로 인식 → 판매자 유리 | 상대가 조정: "$100만? 너무 높다. $70만으로 해줄래?" → "아니면 $85만?" |
| **구매자 입장** | "이 가격은 $40만이 적정입니다" (실제: $60만 시장가) | 판매자가 $40만을 "저가"로 인식 → 구매자 유리 | 판매자가 조정: "$40만? 불가능. $70만은 필요" → "그럼 $65만?" |

### 앵커링의 2단계 프로세스

```
1. Generate Anchor (앵커 생성)
   → 첫 제안이 모든 판단의 "기준점"이 됨

2. Insufficient Adjustment (조정 부족)
   → 상대가 앵커에서 벗어나려 하지만, 심리적으로 완전히 벗어나지 못함
   → 상대가 "합리적"이라고 생각하는 "조정 폭"도 앵커 근처에 머무름
```

### 앵커링 강도 요인

| 요인 | 영향 | 대응 |
|------|------|------|
| **앵커의 극단성** | 극단적일수록 강함 (하지만 신뢰도 ↓) | 근거 제시 필수 (시장 벤치마크, 선례) |
| **상대의 전문성** | 낮을수록 앵커링 효과 강함 | 상대가 초보자면 신뢰할 수 있는 객관 기준 제시 필수 |
| **시간 압박** | 시간이 부족할수록 앵커링 효과 강함 (충분히 조정할 여유 없음) | 여유 시간이 있으면 상대의 조정폭 증가 |

### 앵커링 대응 기법

| 기법 | 메커니즘 | 예 |
|------|---------|------|
| **동시 공개 (Simultaneous Reveal)** | 양측이 동시에 제안 공개 (순차 제시 금지) | "3, 2, 1... 동시에 제시합시다" → 상대의 앵커링 영향 감소 |
| **Counter-Anchoring** | 상대 앵커 직후 즉시 반대 앵커 제시 | 상대: "$100만" / 우리: "아니, $40만이 시장가입니다" → 앵커 충돌 → 상대 앵커 약화 |
| **다중 기준점** | 객관 기준 여러 개 제시 | "시장 벤치마크는 $70~80만, 역사적 거래는 $75만, 우리 비용은 $65만입니다" → 단일 앵커 우위 약화 |
| **앵커의 정당성 공격** | 상대 앵커의 논리 흠집 | 상대: "$100만이 시장가" / 우리: "그 시장가는 누구 기준인가요? 우리가 확인한 건 $75만입니다" |
| **앵커 무시** | 앵커에 반응 안 함 (고급 기법) | 상대: "$100만" / 우리: "자세하게 설명해주시겠어요?" → 앵커 넘어 실제 대화로 진행 |

## Reactive Devaluation — 반응 평가절하

### 메커니즘 (Lee Ross)

**실험 (Israeli-Palestinian context)**:
- 그룹 A: "제안: 팔레스타인이 자치권 획득 + 이스라엘 안보 보장" (중립자 제시)
- 그룹 B: 똑같은 제안을 "이스라엘이 제안"한 것으로 설명

→ 그룹 A가 같은 제안을 더 호의적으로 평가. 그룹 B는 "이스라엘이 제안했으니 함정이 있을 것"이라고 의심.

### 협상에서의 적용

| 상황 | 반응 평가절하 | 신호 |
|------|------------|------|
| **상대가 제안을 약간 양보함** | "상대가 양보했다는 건, 그 조건이 사실 상대에게 이득이라는 뜻이겠지" | 상대가 좋은 제안을 해도 의심 |
| **상대가 "이건 좋은 거다"라고 강조** | "상대가 강조한다는 건 자신에게 유리하다는 뜻이겠지" | 긍정 강조가 역효과 |
| **상대가 먼저 이해관계 공개** | "상대가 먼저 이해관계를 말했다는 건, 우리를 속이려는 수작이겠지" | 투명성이 신뢰로 이어지지 않음 |

### 반응 평가절하의 4가지 메커니즘

| 메커니즘 | 설명 | 사례 |
|---------|------|------|
| **Inference from acceptance** | 상대가 받아들였다는 건 상대에게 이득이라는 추론 | "그 가격으로 수용했다는 건 상대도 수익이 난다는 뜻이겠지" |
| **Opportunity cost** | 상대가 받아들인 조건은, 더 나은 선택지를 포기한 거 아닐까? | "상대가 이 조건을 선택했다는 건, 다른 옵션보다 낫다는 뜻이겠지" |
| **Psychological resistance** | 상대 요구를 받아들이고 싶지 않은 심리 | "상대가 제안하니까 반발하고 싶다" |
| **Zero-sum belief** | 협상은 순수 분배 (one wins, one loses)라는 고정관념 | "상대가 얻으면, 나는 잃는다" |

### 반응 평가절하 대응 기법

| 기법 | 메커니즘 | 효과 |
|------|---------|------|
| **제3자 재제시** | 우리 제안을 상대가 다시 제시하게 하기 | 상대 제안이 아니라 "상대가 생각해낸 아이디어"로 프레이밍 → 평가절하 감소 |
| **근거-우선 (Evidence-First)** | 제안 전에 근거부터 설명 | "시장 벤치마크는 $80만이고, 그 기준에 따르면 우리 제안은 $75만입니다" → 제안이 논리의 결과라고 인식 |
| **상대 입장에서 공동 도출** | 상대와 함께 조건을 도출하게 하기 | "당신의 필요를 고려하면, 어떤 조건이 공정할까요?" → 상대가 스스로 제안. 평가절하 없음 |
| **평가절하 명시적 인정** | "이 제안에 대해 의심스러울 수 있지만..." | 상대의 의심을 먼저 인정 → 의심의 정당성 인정. 신뢰 역설 (Trust Paradox) |

## Endowment Effect — 보유 효과

### 정의

같은 물건이라도, 자신이 소유한 것은 소유하지 않은 것보다 더 가치 있다고 평가.

### 협상에서의 의미

| 상황 | 영향 | 문제 |
|------|------|------|
| **판매자** | 자신의 제품·자산을 시장가보다 높게 평가 | "우리 회사는 $200만 이상이어야 팔아" (시장가: $150만) → Overprice → 매각 어려움 |
| **구매자** | 지금까지 사용한 시스템·프로세스를 높게 평가 | "지금 시스템을 바꾸면 안 돼" (더 나은 시스템 가능) → 혁신 거부 |
| **근속 직원** | 현재 직책·역할을 높게 평가 | "이 팀에서 내 위치는 소중하니까, 이동 제안은 거부" → 경직된 인사 운영 |

### WTA vs WTP (Willingness to Accept vs Willingness to Pay)

```
시장가: $100만

판매자 입장 (WTA):
- "이 자산 팔겠습니다" → 최소 $130만 (endowment effect)

구매자 입장 (WTP):
- "이 자산 사겠습니다" → 최대 $70만 (endowment effect)

→ Gap: $60만. 합의 거의 불가능 (ZOPA 부재)
```

### Endowment Effect 극복

| 기법 | 메커니즘 | 예 |
|------|---------|------|
| **External Reference Frame** | 시장 기준, 객관 평가 도입 | "M&A 회사들이 평가한 기준은 $160만입니다" (자신의 주관 감정 제외) |
| **Loss Framing** | 현재 보유물을 "잃을 것"으로 프레이밍 | "이 시스템을 안 바꾸면, 경쟁사에 3년 안에 추월당합니다" → 손실 회피 심리 자극 (다음 참고) |
| **Brief Ownership Period** | 소유 기간 강조 ("아직 얼마 안 안 했잖아") | "최근 인수하셨으니까, 지금 바꾸셔도 손실 크지 않습니다" |
| **Reference Prices Multiple** | 기준 가격을 여러 개 제시 | "$100만뿐만 아니라, 대체가(Replacement cost)는 $90만, 시장 comparable은 $95만입니다" |

## Loss Aversion — 손실 회피

### 정의 (Kahneman & Tversky)

같은 액수라도, **이득의 기쁨 < 손실의 고통**.

정확히: **손실 회피율 = 약 2:1** (즉, $100 잃는 고통 ≈ $200 버는 기쁨)

### 협상에서의 적용

| 프레이밍 | 호응 |
|---------|------|
| **Gain Frame** (이득 강조): "이 계약을 맺으면 연 $50만의 이익이 생깁니다" | 약한 호응 |
| **Loss Frame** (손실 강조): "이 계약을 맺지 않으면 연 $50만의 기회상실입니다" | 강한 호응 (2배 효과) |

### Loss Aversion 전술

| 전술 | 메커니즘 | 예 |
|------|---------|------|
| **Opportunity Loss** | "지금 안 하면 시장 기회 상실" | "3개월 후에 이 기술은 시장에서 outdated될 것 같습니다. 지금이 기회입니다" |
| **Relationship Loss** | "협상 결렬 = 관계 손상" | "만약 이 협상이 깨지면, 향후 함께 일할 기회도 없어집니다" |
| **Status Quo Loss** | "현상 유지는 사실 손실" | "매년 3% 성장만 하면 5년 뒤엔 경쟁사에 추월당합니다" |
| **Regulatory Loss** | "규제 변화로 인한 손실" | "2년 뒤 법규가 바뀌면, 지금 수용할 수 있는 조건도 불가능해질 겁니다" |

### 주의

- 과도한 loss framing은 상대를 "협박당한다"고 느끼게 함 → 협상 결렬
- Gain frame과 조합: "이득도 있고, 손실도 피할 수 있는 구조"로 제시

## Fairness Norms & Reciprocity — 공정성과 상호성

### 공정성 규범의 강력함

**Ultimatum Game 실험**:
```
제안자: 10,000원을 두 사람이 나눔. 
"당신이 8,000원, 내가 2,000원 가져가겠습니다"

응답자 선택지:
A. 수용 (자신은 2,000원 得)
B. 거부 (제안자도 못 가지고 모두 0원)

이성적 예측: 응답자는 수용 (2,000원이라도 0원보다 낫다)

현실: 응답자 거부율 50%. "공정하지 않은 제안을 수용할 바엔 제안자도 벌받겠다"
```

→ **공정성은 경제적 이성을 이긴다.**

### 협상에서의 공정성

| 상황 | 공정성 요구 | 대응 |
|------|-----------|------|
| **가격 협상** | "왜 우리만 양보해야 해? 상호 양보가 공정하지" | 기준을 먼저 정하기: "시장가 기준으로 공정하게 50:50 나누자" → 그 기준에 따라 분배 |
| **이익 분배** | "우리도 노력 들었는데, 우리 몫이 없다" | 공헌도 명시: "당신들이 개발 50%, 우리가 마케팅 50% 했으니 50:50 분배가 공정" |
| **부담금 분배** | "왜 우리가 더 내야 해?" | 기여도 기준 공개: "우리가 마케팅 비용 60% 담당하니까, 이익도 60% 가져가는 게 공정" |

### Reciprocity (상호성) — Cialdini의 6가지 원칙

**규칙**: 상대가 양보하면, 우리도 양보할 의무가 생김. 위반 시 강한 거부감.

| 단계 | 메커니즘 | 효과 |
|------|---------|------|
| **1. 우리가 먼저 양보** | "우리가 먼저 신뢰 신호를 보냅니다" | 상대에게 "우리도 양보해야 한다"는 심리적 의무 생성 |
| **2. 양보의 크기 명시** | "우리는 이 부분에서 $20만을 양보했습니다" | 상대가 정확히 인지 → 상호성 작동 |
| **3. 상대 양보 요청** | "이제 당신 차례예요" (직접 말 금지, 암시) | 상대 자발적 양보 유도 |
| **4. 상호 양보 순환** | 우리 양보 → 상대 양보 → 우리 양보 → ... | 선순환. 신뢰 구축. 합의로 수렴 |

### Reciprocity 위반의 파괴력

```
우리가 양보 → 상대는 안 하고 더 요구
→ 상대가 "reciprocity rule을 위반했다"고 인식
→ 상대에 대한 신뢰 완전 붕괴
→ 강한 보복 심리 (상대도 손해를 감수하면서 우리에게 피해)
```

→ **"호혜"를 명시적으로 표현해야 함**: "우리가 이 부분 양보했으니, 당신도 이 부분 양보해주실 수 있을까요?"

## Psychology Screening Protocol

협상 전, 상대방의 심리 프로필을 스크리닝.

| 심리 메커니즘 | 신호 감지 방법 | 약점 판정 (강/중/약) | 전술 조정 |
|---|---|---|---|
| **Anchoring** | "첫 제안에 즉각 반응하는가? 쉽게 수용하는가?" | 쉽게 수용 = 강한 앵커링 취약 | 공격적 앵커링 활용. 근거 강조 |
| **Reactive Devaluation** | "우리 제안에 항상 의심적인가?" | 의심적 = 강함 | 제3자 재제시. 근거-우선 프레이밍 |
| **Endowment Effect** | "현재 상태 유지를 강하게 강조하는가?" | 강조 빈번 = 강함 | Loss frame 활용. 시장 기준 도입 |
| **Loss Aversion** | "손실을 이득보다 더 강하게 반응하는가?" | 손실 언급에 즉각 반응 = 강함 | Loss frame 강조. 기회상실 강조 |
| **Fairness Norms** | "공정성·상호성을 자주 언급하는가?" | 언급 빈번 = 강함 | 공정성 기준 먼저 합의. 상호 양보 명시 |

### 복합 프로필 예

```
상대: 앵커링 취약 + Loss Aversion 강함 + Fairness 민감

전략:
1. 공격적 앵커링 ("시장가는 $70만입니다" - 첫 제안에서 정함)
2. Loss frame ("지금 이 기회를 놓치면 3개월 뒤엔 불가능합니다")
3. 공정성 기준 명시 ("공정성을 위해, 우리도 이 부분 양보합니다")

→ 상대의 심리 메커니즘 3개를 동시에 자극하여 수용 확률 증가
```
